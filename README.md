# 九九财富投资晨报 📈

**九九** — 老板的超级智能财富与投资助手，每日早 8:00 自动生成并推送的财富投资晨报。

> 🌐 **在线浏览：** https://corinwe.github.io/wealth-invest-daily/

## 目录结构

```
reports/              # 每日晨报 Markdown 文件（YYYY-MM-DD.md）
index.html            # 首页（晨报列表，支持搜索）
report.html           # 晨报详情页（支持上下期导航）
build.py              # 自动从 Markdown 生成静态 HTML
```

## 每日内容

- 📊 持仓个股追踪（阿里、小米、美团、MSFT、ECL、比亚迪、美的）
- 🤖 AI/科技板块追踪（NVDA、AVGO、AMD 等）
- 📈 大盘概览（美股、港股、A股）
- 📌 当日重点关注事件

> 由 [Hermes Agent](https://hermes-agent.nousresearch.com) + DeepSeek 自动生成，每日 8:00 更新。
